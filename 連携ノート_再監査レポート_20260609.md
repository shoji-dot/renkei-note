## 残存する未対応項目

### 小（すぐ対応可能）

**① フィードバック削除機能**
フィードバック（ProductFeedback）のみ削除エンドポイントが未実装。
誤入力・テスト投稿が残り続ける。

**② Materialモデルのデッドコード**
`schema.prisma` に `Material` モデルが定義されているが、対応する画面・APIが存在しない。
→ 使わないなら削除（DBテーブルごと）、使うなら材料管理画面を作る。

### 中（設計判断が必要）

**③ 製造カレンダーの編集機能**
削除は実装済みだが、登録後に内容（作業内容・人数・メモ）を修正できない。
誤字や人数変更の際は削除→再登録が必要。

### 将来検討（変更なし）

- 通知機能（LINE Messaging API または PWA Web Push）
- 検索機能（Prisma `contains` で実装可能）
- 未読バッジ
- CSV出力
